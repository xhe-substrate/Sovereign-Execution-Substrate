# Sovereign Execution Substrate (SES) v2.0

> Distributed execution platform with P2P networking and automatic value attribution

## Quick Start

### Browser (Instant)
```bash
open index.html
```

### Desktop App
```bash
npm install
npm run dev
```

### VS Code Extension
```bash
cd vscode-extension
npm install
code .
# Press F5 to launch
```
