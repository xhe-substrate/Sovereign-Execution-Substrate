# SES Context OS - VS Code Extension

Automatic context capture for zero-cost context switching.

## Installation
```bash
npm install
```

## Development

Press F5 in VS Code to launch Extension Development Host.

## Usage

- `Cmd+Shift+S` - Capture current context
- `Cmd+Shift+C` - Switch context
- Command Palette: "SES: ..." commands

## Features

- Automatic context capture
- Breakpoint preservation
- Task/terminal state saving
- Git branch tracking
